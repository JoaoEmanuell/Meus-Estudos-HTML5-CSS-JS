==================================
Tags:

DOCTYPE = Define o conteudo (html)

meta = configurações geral

title = Titulo da pagina (nome da pagina)

h1 = Titulo, ou seja um nome grande

hr = desenha uma linha horizontal

p = Paragrafo
==================================
Codigo:

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu primeiro exercicio</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <hr>
    <p>Esse é o meu primeiro documento HTML!
        Estou muito feliz!
    </p>
    <p>Esté é um momento unico, estou criando um site</p>
</body>
</html>
